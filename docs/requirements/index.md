# 要件定義

## ビジネスの背景

### 背景

記事を公開するブログプラットフォームを作成する

### アクター

- 著者
- 閲覧者

### 外部システム

なし

### 要求

記事の執筆、公開が億劫にならないような手軽なインターフェースがほしい

### 制約

個人向けのシステムにする

### ビジネスポリシー

- GitHub Flavored Markdown で記事を作成する
- 記事をバージョン管理し、任意のバージョンを閲覧できるほか任意のバージョン同士の差分も確認できる

## 概要

- 執筆
    - 著者は記事を執筆する
- 閲覧
    - 閲覧者は記事を閲覧する
